# RobustTrainer

This repository contains code for the paper "Robust Learning of Deep Predictive Models from Noisy and Imbalanced Software Engineering Datasets"

## Outline

The contents of this repository are as follows:
- [/code](https://github.com/RobustTrainer/RobustTrainer/tree/main/code) contains the source code for RobustTrainer
- [/results](https://github.com/RobustTrainer/RobustTrainer/tree/main/results) contains the raw data from our experiments
- [/datasets](https://github.com/RobustTrainer/RobustTrainer/tree/main/datasets) contains datasets used in our experiments
